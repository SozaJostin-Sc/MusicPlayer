# 🎵 Blue - Simple Music Player

This is a **minimalist music player** with an image carousel and progress bar, built using **HTML, CSS, and JavaScript**.

## 🚀 Features

✅ **Automatic image carousel**  
✅ **Audio player** with control buttons  
✅ **Progress bar** displaying song progress  
✅ **Elegant and responsive interface** with smooth animations  
✅ **"Like" button** with a heart icon  
✅ **QR code** for sharing

## 🛠 Technologies Used

- HTML5
- CSS3 (Flexbox, Animations, Gradients)
- JavaScript (Audio handling and DOM manipulation)

## 📂 Project Structure
```

📁 Project  
│── 📄 index.html # Player structure  
│── 📄 style.css # Styles and animations  
│── 📄 script.js # Player functionality  
│── 📁 assets/ # Images, songs, and resources  
│── 📄 README.md # Project documentation

```

## 🎮 How to Use

1️⃣ Clone this repository or download the files.
2️⃣ Open `index.html` in a web browser.
3️⃣ Enjoy the music and image carousel.

## 🎨 Customization

🔹 Add more songs in `script.js` inside the `songs` array.
🔹 Change the carousel images in `index.html`.
🔹 Modify the styles in `style.css` to fit your design preferences.

This project is open-source, and you are free to modify it.
